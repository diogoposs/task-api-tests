# 🧪 Task API Tests

Automated API testing project for the Task API application.

This project validates REST endpoints using BDD with Cucumber and Rest Assured.

---

## 🚀 Technologies

- Java 17
- Maven
- Rest Assured
- Cucumber
- JUnit 5

---

## 📌 Test Coverage

- Create Task (POST)
- Get Task by ID (GET)
- Update Task (PUT)
- Delete Task (DELETE)
- Status code validation
- Response body validation

---

## ▶️ How to Run

Make sure the Task API is running locally.

Run:

mvn test
